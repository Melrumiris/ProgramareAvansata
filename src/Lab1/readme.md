## Finished Compulsory, Homework and Advanced
>
>### Argument types for the Homework main method:
>

>[matrixSize] cricle [circleCenterX] [circleCenterY] [radius]  
> - Example: 100 rectangle 25 25 20 

>[matrixSize] rectangle [rectangleCornerX] [rectangleCornerY] [rectangleWidth] [rectangleHeight]  
> - Example: 100 rectangle 20 20 25 30

>Writing 'test' before the other arguments will print the execution time instead of the matrix.  
> - Example: test 30000 rectangle 200 200 250 205
> (Note: If I run it with n = 35000, it gives the error: Requested array size exceeds VM limit regardless of allocated heap space)
 
>Writing 'bounded' after the other arguments will also print the details about the bounding box.  
> - Example: 100 circle 50 50 30 bounded
